MA RENT:

Full Stack Django Web App for Accomodation.


Local deployment:

1- Open project
2- create a virtual environement (mac) : python3 -m venv myenv
3- activate it (mac): source myenv/bin/activate
4- To install the packages listed in a requirements.txt file: pip install -r requirements.txt
5- Run the App : pyhton manage.py runserver
